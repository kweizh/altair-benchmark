**What the script does:**

```python
import altair as alt
from vega_datasets import data

chart = (
    alt.Chart(data.iris.url)          # URL-based data source
    .mark_bar()
    .encode(
        x=alt.X("petalLength:Q", bin=alt.Bin(maxbins=20)),  # binned, quantitative
        y=alt.Y("count()"),                                   # count aggregate
        color=alt.Color("species:N"),                         # nominal color → stacked by default
    )
)

chart.save("chart.html")
```
